Detailed Description

Display one or more of Langton's Ant!

Contains the data and method needed in order to hold and move around one or more LangtonAnt objects.

You can have one LangtonAnt, or multiple.

You can move them in parallel, or in sequence.

You can also shade the colors of each LangtonAnt.
