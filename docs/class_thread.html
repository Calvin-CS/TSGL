Detailed Description

The Thread class encapsulates the details of creating a pthread.

Thread.h contains the abstract class necessary in order to have pthreads for Producers and Consumers. Adapted from: http://jrdodds.blogs.com/blog/2006/06/encapsulating_a.html https://slworkthings.wordpress.com/2009/11/10/a-pthread-wrapper-class-part-1/ https://cppcodetips.wordpress.com/2013/12/05/pthread-c-wrapper/ http://peter.bourgon.org/blog/2010/10/27/who-needs-boost-a-simple-pthreads-wrapper.html Practical Guide To Pthread Programming in C++ By Swaminathan Bhaskar (pdf) The Thread class encapsulates the details of creating a pthread. It is abstract; cannot be instantiated.
