Detailed Description

A lock giving equal priority to Readers and Writers.

FLock.h provides the fair monitor for the Reader-Writer visualization that gives equal preference to the Readers and Writers. This class is a subclass of Lock and implements its locking and unlocking virtual methods. implements the "commonly known solution" presented in: https://arxiv.org/pdf/1309.4507.pdf

Inheritance: Lock class.

Implements the locking and unlocking methods of a monitor.
