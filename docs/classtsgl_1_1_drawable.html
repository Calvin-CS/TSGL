Detailed Description

A class for drawing objects onto a Canvas or CartesianCanvas.

Warning
Though extending this class must be allowed due to the way the code is set up, attempting to do so could potentially mess up the internal GL calls the library uses. Proceed with great caution.

Drawable provides a base class for drawing 3D objects to a Canvas or CartesianCanvas.

Note
Drawable is abstract, and must be extended by the user.

vertices should be an array of floating point values in TSGL's vertex format.

numberofvertices should be the actual integer number of vertices to be drawn (e.g., 36 for a cube).

geometryType should be one of GL's primitive drawing modes. See https://www.opengl.org/sdk/docs/man2/xhtml/glBegin.xml for further information.

Theoretically, you could potentially extend the Drawable class so that you can create another Drawable class that suits your needs.

However, this is not recommended for normal use of the TSGL library.
