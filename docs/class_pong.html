Detailed Description

An old-school classic!

Draw the interactive game of Pong. The two paddles are objects and the ball is also an object.

The constructor constructs the two paddle objects and the ball object.

Use the W and S keys in order to move the left paddle, the Up and Down arrow keys in order to move the right paddle.

Everything else is handled in the draw() method (button bindings, score keeping, drawing the objects and the game itself, ball collisions with the paddles and the boundaries).
