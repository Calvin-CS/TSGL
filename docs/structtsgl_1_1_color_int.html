Detailed Description

Integer RGBA color struct.

ColorInt defines a color with integer red, green, blue, and alpha components.

Parameters
RRed component, between 0 and 255 inclusive.
GGreen component, between 0 and 255 inclusive.
BBlue component, between 0 and 255 inclusive.
AAlpha component, between 0 and 255 inclusive.
