Detailed Description

Displays a spectrogram of a given data set.

Spectrogram is a class for visualizing data as a color spectrum. This data will typically be hues, but Spectrogram will accept any data mapping to an integer value between 0 and 255.

Updates a spectrogram with new data, using critical sections for thread safety.

This function adds the value of weight to the hue specified by index, and adds the value (decay^n)* weight to all hues n steps away from index.

Parameters
indexIndex of the hue to update. Value is taken mod 256.
weightThe value to add to index.
decayFalloff for weight upon adjacent values.
