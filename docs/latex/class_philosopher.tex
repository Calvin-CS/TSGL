\doxysubsection{Detailed Description}
Object representing a philosopher in the Dining Philosophers\textquotesingle{} problem. 

The \mbox{\hyperlink{class_philosopher}{Philosopher}} class contains variables and methods necessary for representing a philosopher at a table. Each \mbox{\hyperlink{class_philosopher}{Philosopher}} may acquire or release the fork to his left or to his right (or both), with his state changing accordingly.
