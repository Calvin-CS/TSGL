\doxysubsection{Detailed Description}
A lock giving priority to Writers in the visualization. 

\mbox{\hyperlink{_w_lock_8h_source}{W\+Lock.\+h}} provides the monitor for the Reader-\/\+Writer visualization that gives preference to the Writers. This class is a subclass of \mbox{\hyperlink{class_lock}{Lock}} and implements its locking and unlocking virtual methods.
