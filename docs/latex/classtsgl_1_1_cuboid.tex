\doxysubsection{Detailed Description}
Draw an arbitrary \mbox{\hyperlink{classtsgl_1_1_cuboid}{Cuboid}} with colored vertices. 

\mbox{\hyperlink{classtsgl_1_1_cuboid}{Cuboid}} is a class for holding vertex data for a \mbox{\hyperlink{classtsgl_1_1_cuboid}{Cuboid}}.

\mbox{\hyperlink{classtsgl_1_1_cuboid}{Cuboid}} is a 6-\/sided subclass of \mbox{\hyperlink{classtsgl_1_1_shape}{Shape}} with all rectangular faces.
