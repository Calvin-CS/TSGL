\doxysubsection{Detailed Description}
Draw an arbitrary triangle strip with colored vertices. 

\mbox{\hyperlink{classtsgl_1_1_triangle_strip}{Triangle\+Strip}} is a class for holding vertex data for a triangle strip with colored vertices.

Vertices are drawn in triangle strip format, where the first three vertices make up the first triangle, the next vertex plus the previous two make up the second triangle, and so on.

This method is optimized for long lists and offers a marked improvement over drawing individual \mbox{\hyperlink{classtsgl_1_1_triangle}{Triangle}} instances. \begin{DoxyNote}{Note}
The \mbox{\hyperlink{classtsgl_1_1_shape_ac0ea13ecde43ce62262f82597fb94755}{add\+Vertex()}} method must be called the same number of times as specified in the constructor. 

Calling \mbox{\hyperlink{classtsgl_1_1_shape_ac0ea13ecde43ce62262f82597fb94755}{add\+Vertex()}} after all vertices have been added will do nothing. 

Calling \mbox{\hyperlink{classtsgl_1_1_shape_ae0905e73f1652d92cef9c7f2c38572f3}{draw()}} before all vertices have been added will do nothing. 
\end{DoxyNote}
