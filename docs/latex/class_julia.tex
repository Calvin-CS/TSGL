\doxysubsection{Detailed Description}
Draw a \mbox{\hyperlink{class_julia}{Julia}} set. 

Contains all of the information necessary in order to draw a \mbox{\hyperlink{class_julia}{Julia}} set onto a Cartesian\+Canvas.

Can zoom in and out of the \mbox{\hyperlink{class_julia}{Julia}} set by scrolling the mouse wheel up and down (respectively).

Same is true when clicking the left mouse button (zooming in) and the right mouse button (zooming out).

Can also clear the screen by pressing the spacebar.
