\doxysubsection{Detailed Description}
Draws a \mbox{\hyperlink{class_gradient_mandelbrot}{Gradient\+Mandelbrot}} set on a Cartesian\+Canvas. 

Similar to a normal \mbox{\hyperlink{class_mandelbrot}{Mandelbrot}} set but with a smoother shading.

Extends the \mbox{\hyperlink{class_mandelbrot}{Mandelbrot}} class and overrides its \mbox{\hyperlink{class_gradient_mandelbrot_a1d4aa3e44d7d1c2241545b60c79985df}{draw()}} function and set\+Redraw() function.

Can zoom in and out of the \mbox{\hyperlink{class_mandelbrot}{Mandelbrot}} set by scrolling up and down on the mouse wheel (respectively). \begin{DoxySeeAlso}{See also}
