\doxysubsection{Detailed Description}
Provides a a tool for computing and visualizing integrals of functions. 

\mbox{\hyperlink{classtsgl_1_1_integral_viewer}{Integral\+Viewer}} provides a simple interface for integrating functions and outputting the results of the integration, both numerically and visually. \mbox{\hyperlink{classtsgl_1_1_integral_viewer}{Integral\+Viewer}} can evaluate an arbitrary function of the type {\ttfamily Decimal my\+Function(\+Decimal x)}, where x is the input value of the function and the return value is the y value of the function. \mbox{\hyperlink{classtsgl_1_1_integral_viewer}{Integral\+Viewer}} can compute integrals using one or both of the rectangle method and the trapezoid method, and provides functions for displaying both on a custom \mbox{\hyperlink{classtsgl_1_1_canvas}{Canvas}}. Furthermore, these computations and visualizations are thread-\/safe; the number of threads to use can be set with {\ttfamily omp\+\_\+set\+\_\+num\+\_\+threads()}.
