\doxysubsection{Detailed Description}
Draw an arbitrary \mbox{\hyperlink{classtsgl_1_1_cube}{Cube}} with colored vertices. 

\mbox{\hyperlink{classtsgl_1_1_cube}{Cube}} is a class for holding vertex data for a \mbox{\hyperlink{classtsgl_1_1_cube}{Cube}}.

\mbox{\hyperlink{classtsgl_1_1_cube}{Cube}} is a 6-\/sided subclass of \mbox{\hyperlink{classtsgl_1_1_shape}{Shape}} with all square faces.
