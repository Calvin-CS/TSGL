\doxysubsection{Detailed Description}
Draw multiple lines chained together. 

\mbox{\hyperlink{classtsgl_1_1_polyline}{Polyline}} is a class for holding vertex data for multiple lines whose endpoints are connected.

This method is optimized for long lists and offers a marked improvement over drawing individual \mbox{\hyperlink{classtsgl_1_1_line}{Line}} instances. \begin{DoxyNote}{Note}
The \mbox{\hyperlink{classtsgl_1_1_polyline_a938427299b0a5ecef0bbc4cf38d5ee1c}{add\+Vertex()}} method must be called the same number of times as specified in the constructor, unless a float array is also passed. 

Calling \mbox{\hyperlink{classtsgl_1_1_polyline_a938427299b0a5ecef0bbc4cf38d5ee1c}{add\+Vertex()}} after all vertices have been added will do nothing. 

Calling Drawable\+::draw() before all vertices have been added will do nothing. 
\end{DoxyNote}
