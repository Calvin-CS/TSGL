\doxysubsection{Detailed Description}
Draw a string of text. 

\mbox{\hyperlink{classtsgl_1_1_text}{Text}} is a class for holding the data necessary for rendering a string of text. \begin{DoxyNote}{Note}
\mbox{\hyperlink{classtsgl_1_1_text}{Text}} is aligned by the upper-\/left corner. 

Fonts supported by Free\+Type are also supported. 
\end{DoxyNote}
