Detailed Description

A base class for creating mathematical functions plottable by a CartesianCanvas.

Function provides a base class for the creation of mathematical functions. By extending this class and overriding the valueAt() method, users can easily plot the values of their function on a CartesianCanvas.

A number of pre-built Function subclasses are included in the header file for reference.
