Detailed Description

Floating point RGBA color struct.

ColorFloat defines a color with floating point red, green, blue, and alpha components.

Parameters
RRed component, between 0 and 1 inclusive.
GGreen component, between 0 and 1 inclusive.
BBlue component, between 0 and 1 inclusive.
AAlpha component, between 0 and 1 inclusive.
