Detailed Description

Draw a Background for the Canvas with colored pixels.

Background is a class for holding colored pixel data.

Constructor & Destructor Documentation

Explicitly constructs a new Background.

Explicit constructor for a Background object.

Parameters
widthBackground's width in pixels.
heightBackground's height in pixels.
cA ColorFloat for the Background's original color.
Warning
An invariant is held where if width or height isn't positive then an error message is given.
Returns
A new Background to which Drawables and Pixels can be drawn procedurally.

◆ draw()

void tsgl::Background::draw ( )
virtual

Draw the Background.

This function actually draws the Background to the Canvas.

Note
On each draw cycle, first any Drawables that have been newly added to the Background will be rendered, and then any new calls to drawPixel will be processed.

Reimplemented in tsgl::CartesianBackground.
