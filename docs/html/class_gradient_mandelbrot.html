Detailed Description

Draws a GradientMandelbrot set on a CartesianCanvas.

Similar to a normal Mandelbrot set but with a smoother shading.

Extends the Mandelbrot class and overrides its draw() function and setRedraw() function.

Can zoom in and out of the Mandelbrot set by scrolling up and down on the mouse wheel (respectively).
