\doxysubsection{Detailed Description}
Draws a \mbox{\hyperlink{class_nova}{Nova}} (Newton Fractal) set on a Cartesian\+Canvas. 

Extends the \mbox{\hyperlink{class_mandelbrot}{Mandelbrot}} class and overrides its \mbox{\hyperlink{class_nova_a66935ba0814dfabcae2481c128c336ec}{draw()}} function and set\+Redraw() function.
