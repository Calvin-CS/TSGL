\doxysubsection{Detailed Description}
An old-\/school classic! 

Draw the interactive game of \mbox{\hyperlink{class_pong}{Pong}}. The two paddles are objects and the ball is also an object.

The constructor constructs the two paddle objects and the ball object.

Use the W and S keys in order to move the left paddle, the Up and Down arrow keys in order to move the right paddle.

Everything else is handled in the \mbox{\hyperlink{class_pong_a3c7242248be0e6980595faaa0a927866}{draw()}} method (button bindings, score keeping, drawing the objects and the game itself, ball collisions with the paddles and the boundaries). \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_paddle}{Paddle}} class, \mbox{\hyperlink{class_ball}{Ball}} class. 
\end{DoxySeeAlso}
