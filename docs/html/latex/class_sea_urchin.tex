\doxysubsection{Detailed Description}
Who doesn\textquotesingle{}t love sea urchins? 

Draws a sea urchin onto the Canvas.

Used as an example of what it means to put a process on a thread.

The \mbox{\hyperlink{class_sea_urchin}{Sea\+Urchin}} objects are created in a parallel block, and each thread gets one \mbox{\hyperlink{class_sea_urchin}{Sea\+Urchin}}.

Then, the thread draws the \mbox{\hyperlink{class_sea_urchin}{Sea\+Urchin}} onto the Canvas.

Each \mbox{\hyperlink{class_sea_urchin}{Sea\+Urchin}} object has its own color and plot data based off of the thread\textquotesingle{}s id number.

\mbox{\hyperlink{class_sea_urchin}{Sea\+Urchin}} objects are drawn similar to the line fan in test\+Line\+Fan. \begin{DoxySeeAlso}{See also}
test\+Line\+Fan. 
\end{DoxySeeAlso}
