\doxysubsection{Detailed Description}
How can you hit a ball without a paddle? 

class \mbox{\hyperlink{class_paddle}{Paddle}}

Creates a \mbox{\hyperlink{class_paddle}{Paddle}} in order to play the game, \mbox{\hyperlink{class_pong}{Pong}}.

The W and S keys move the left paddle, the Up and Down arrow keys move the right paddle in the game.

Collisions with the ball are not handled here; they are handled in the \mbox{\hyperlink{class_pong}{Pong}} class\textquotesingle{} draw() method.

The left and right paddles in the game are created from this class; they are essentially the same object. However, there are side parameters in some of the methods that determine which side to place the \mbox{\hyperlink{class_paddle}{Paddle}} object on and how to treat the \mbox{\hyperlink{class_paddle}{Paddle}} object (either as the left or the right paddle in the game). The \mbox{\hyperlink{class_pong}{Pong}} class handles how to distribute points to the \mbox{\hyperlink{class_paddle}{Paddle}} objects in the game (the \mbox{\hyperlink{class_paddle}{Paddle}} class only has an \mbox{\hyperlink{class_paddle_a05d45fe2ca65d7baade4162c56a478f0}{increment()}} method that increments its score counter; the \mbox{\hyperlink{class_pong}{Pong}} class determines which object should call that method when a point is earned). \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_ball}{Ball}} class, \mbox{\hyperlink{class_pong}{Pong}} class. 
\end{DoxySeeAlso}
