\doxysubsection{Detailed Description}
Displays a spectrogram of a given data set. 

\mbox{\hyperlink{classtsgl_1_1_spectrogram}{Spectrogram}} is a class for visualizing data as a color spectrum. This data will typically be hues, but \mbox{\hyperlink{classtsgl_1_1_spectrogram}{Spectrogram}} will accept any data mapping to an integer value between 0 and 255.
