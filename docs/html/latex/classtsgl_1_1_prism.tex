\doxysubsection{Detailed Description}
Draw an arbitrary \mbox{\hyperlink{classtsgl_1_1_prism}{Prism}} with colored vertices. 

\mbox{\hyperlink{classtsgl_1_1_prism}{Prism}} is a class for holding vertex data for a \mbox{\hyperlink{classtsgl_1_1_prism}{Prism}} with a base with at least 3 sides.
