\doxysubsection{Detailed Description}
A class for various timing operations. 

\mbox{\hyperlink{classtsgl_1_1_timer}{Timer}} provides a simple timer for timing, sleeping threads, and keeping track of the current rendering frame.

Sleeps the \mbox{\hyperlink{classtsgl_1_1_timer}{Timer}}\textquotesingle{}s current thread until its period elapses. 

This function tells the currently executing thread to sleep until the rest of the \mbox{\hyperlink{classtsgl_1_1_timer}{Timer}} instance\textquotesingle{}s remaining period expires.

If the \mbox{\hyperlink{classtsgl_1_1_timer}{Timer}}\textquotesingle{}s period has elapsed since last call, the thread will continue execution normally until the next call to \mbox{\hyperlink{classtsgl_1_1_timer_a5c6e0dca6793d48c675e05b983da7f07}{sleep()}}.
