\doxysubsection{Detailed Description}
A lock giving equal priority to Readers and Writers. 

\mbox{\hyperlink{_f_lock_8h_source}{F\+Lock.\+h}} provides the fair monitor for the Reader-\/\+Writer visualization that gives equal preference to the Readers and Writers. This class is a subclass of \mbox{\hyperlink{class_lock}{Lock}} and implements its locking and unlocking virtual methods. implements the \char`\"{}commonly known solution\char`\"{} presented in\+: \href{https://arxiv.org/pdf/1309.4507.pdf}{\texttt{ https\+://arxiv.\+org/pdf/1309.\+4507.\+pdf}}

Inheritance\+: \mbox{\hyperlink{class_lock}{Lock}} class.

Implements the locking and unlocking methods of a monitor.
