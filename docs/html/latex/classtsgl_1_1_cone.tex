\doxysubsection{Detailed Description}
Draw an arbitrary \mbox{\hyperlink{classtsgl_1_1_cone}{Cone}} with colored vertices. 

\mbox{\hyperlink{classtsgl_1_1_cone}{Cone}} is a class for holding vertex data for a cone.

\mbox{\hyperlink{classtsgl_1_1_cone}{Cone}} is a subclass of \mbox{\hyperlink{classtsgl_1_1_pyramid}{Pyramid}} with a base of (radius $\ast$ 3) sides, minimum of 3.
