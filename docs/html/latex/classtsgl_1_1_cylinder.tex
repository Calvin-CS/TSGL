\doxysubsection{Detailed Description}
Draw an arbitrary \mbox{\hyperlink{classtsgl_1_1_cylinder}{Cylinder}} with colored vertices. 

\mbox{\hyperlink{classtsgl_1_1_cylinder}{Cylinder}} is a class for holding vertex data for a \mbox{\hyperlink{classtsgl_1_1_cylinder}{Cylinder}}.

\mbox{\hyperlink{classtsgl_1_1_cylinder}{Cylinder}} is a subclass of \mbox{\hyperlink{classtsgl_1_1_prism}{Prism}} with a circular base.
