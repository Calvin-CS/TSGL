\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_thread}{Thread}} that writes items in the visualization. 

\mbox{\hyperlink{class_writer}{Writer}} class creates a \mbox{\hyperlink{class_writer}{Writer}} object and inherits from the \mbox{\hyperlink{class_r_w_thread}{R\+W\+Thread}} class.

Inheritance\+: \mbox{\hyperlink{class_r_w_thread}{R\+W\+Thread}} class.

Implements the abstract run() method from the \mbox{\hyperlink{class_thread}{Thread}} class so that the pthread runs the write() method.
