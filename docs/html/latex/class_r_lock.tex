A lock giving priority to Readers. 

\mbox{\hyperlink{_r_lock_8h_source}{R\+Lock.\+h}} provides the monitor for the Reader-\/\+Writer visualization that gives preference to the Readers. This class is a subclass of \mbox{\hyperlink{class_lock}{Lock}} and implements its locking and unlocking virtual methods.
