Detailed Description

Object representing a 3D philosopher in the Dining Philosophers' problem.

The Philosopher3D class contains variables and methods necessary for representing a philosopher at a table. Each Philosopher3D may acquire or release the fork to his left or to his right (or both), with his state changing accordingly.
