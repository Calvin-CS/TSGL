Detailed Description

template<typename Item>
class tsgl::Array< Item >

Custom internal array used by Canvas.

The Array class manages a custom array for storing shapes to be drawn. It contains utility methods for checking if the internal array is empty, methods for emptying and resetting the internal array, and a subscript operator for accessing individual elements.

Note
The Array has wrap-around behavior, behaving similarly to a circular queue.
If a new shape is pushed into a full Array, the first element is deleted and the pointer to the first element is incremented.
