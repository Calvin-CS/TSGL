Detailed Description

Floating point HSVA color struct.

ColorHSV defines a color with floating point hue, saturation, value, and alpha components.

Parameters
HHue component, between 0 and 6 inclusive.
SSaturation component, between 0 and 1 inclusive.
VValue component, between 0 and 1 inclusive.
AAlpha component, between 0 and 1 inclusive.
