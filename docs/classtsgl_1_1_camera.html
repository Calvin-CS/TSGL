Detailed Description

Defines a movable, rotatable camera-like object and its corresponding view matrix.

Camera is used within TSGL::Canvas to define the view matrix.

Parameters
positionThe Camera's location in 3D space, a glm::vec3.
upA 3D vector indicating the up direction in world space.
yawThe camera's yaw rotation (horizontal) in degrees.
pitchThe camera's pitch rotation (vertical) in degrees.
Note
With position.x and position.y equal to 0 and a positive position.z, a yaw of -90 degrees and pitch of 0 degrees will look at (0,0,0).
pitch is constrained between -89 and 89 degrees.
Returns
