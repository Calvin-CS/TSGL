Detailed Description

Draw a procedurally drawn CartesianBackground for the Canvas.

CartesianBackground is a class for holding a framebuffer of color data.

Explicitly constructs a new CartesianBackground.

Creates a Background whose world dimensions don't have to match its pixel dimensions.

Parameters
widthCartesianBackground's width in pixels.
heightCartesianBackground's height in pixels.
xMinx value of the left side of the CartesianBackground in world coordinates.
xMaxx value of the right side of the CartesianBackground in world coordinates.
yMiny value of the bottom side of the CartesianBackground in world coordinates.
yMaxy value of the top side of the CartesianBackground in world coordinates.
cA ColorFloat for the CartesianBackground's original color.
Warning
An invariant is held where if xMax or xMin isn't greater than its respective min then an error message is given.
Returns
A new CartesianBackground to which Drawables and Pixels can be drawn procedurally.

Draw the CartesianBackground.

This function actually draws the CartesianBackground to the Canvas.
